## 8.4 Canada Pension Plan (CPP) - 2026

### CPP Parameters

| Parameter | CPP (except QC) | QPP (QC) |
|:----------|:----------------|:---------|
| **YMPE** (Year's Maximum Pensionable Earnings) | $74,600 | $74,600 |
| **Basic Exemption** | $3,500 | $3,500 |
| **YMCE** (Year's Maximum Contributory Earnings) | $71,100 | $71,100 |
| **Employee Contribution Rate** | 5.95% | 6.30% |
| **Employer Contribution Rate** | 5.95% | 6.30% |
| **Maximum Employee Contribution** | $4,230.45 | $4,479.30 |
| **Maximum Employer Contribution** | $4,230.45 | $4,479.30 |
| **YAMPE** (Year's Additional Maximum Pensionable Earnings) | $85,000 | $85,000 |
| **CPP2/QPP2 Rate** | 4.00% | 4.00% |
| **Maximum CPP2/QPP2** | $416.00 | $416.00 |

### CPP Contribution Formula

```
C = (I - BPE) × 0.0595
```

Where:
- **I** = Gross remuneration for the pay period
- **BPE** = Basic exemption per pay period ($3,500 ÷ P)
- **0.0595** = CPP contribution rate

## 8.8 Tax Credit Rates

### Federal Tax Credit Rate

**All federal non-refundable tax credits** are calculated at:

```
Tax Credit Rate = 14% (0.14)
```

This applies to:
- Basic personal amount
- Spousal amount
- Eligible dependent amount
- Age amount
- Pension income amount
- CPP contributions
- EI premiums
- Canada employment amount
